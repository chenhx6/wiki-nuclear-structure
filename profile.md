# 研究者与知识库档案

## 主要用途

建立面向个人科研工作的专业复利知识库，服务于基于熔合蒸发反应的低能原子核结构研究。

## 当前研究范围与优先级

本知识库服务于实验核物理与低能核结构研究积累，用于支持数据分析、文献问答、论文写作、课题判断和长期知识复用。

A≈130 质量区集体激发研究是当前阶段的重要研究锚点之一，但不是 Wiki 的收录边界。Wiki 不按单一质量区限定收录范围。

优先摄入内容包括：

1. 低能核结构实验文献中可复用的能级结构、带结构、跃迁性质、自旋宇称指定、组态指认、集体运动和竞争解释；
2. 与实验判据和谱学分析方法有关的内容，例如角分布、角关联、DCO 比值、线偏振、ADO、混合比、alignment、spin-parity assignment；
3. 对 wobbling、chirality、shape coexistence、triaxiality、core coupling、configuration assignment 等主题具有比较价值或启发意义的实验案例；
4. 用户明确指定为当前分析、未来课题、数据分析或论文写作需要重点关注的文献、核素、反应体系或实验方法，例如后续基于熔合蒸发反应的目标核谱学研究。

是否建立独立核素页、带结构页、概念页或 project 页，应根据实验核结构相关性、结构信息密度、证据可追溯性、方法复用价值和未来使用价值决定，而不是仅由质量区决定。

注意：本页不记录具体学位论文题目、具体实验反应道或具体核素作为个人经历。

## 期望产出

- 可追溯的论文精读笔记；
- 按核素、带结构和物理现象组织的领域地图；
- 对相互竞争解释的证据矩阵；
- 研究选题、实验分析与论文写作支持；
- 避免重复踩坑的决策与失败记录；
- 能被不同 Agent 和不同模型持续维护的纯文本资产。

## 尚待用户补充

- 常用实验装置、探测阵列和数据分析软件；
- 常用理论模型与计算程序；
- 当前重点核素和在研项目；
- 文献引用格式；
- 对未发表数据、合作材料和审稿内容的保密边界；
- 中文与英文科研写作偏好。
