# Raw：原始证据层

本目录由用户拥有，Agent 只读。

- `papers/`：论文 PDF、补充材料及原始导出；
- `notes/`：外部笔记、会议记录和人工摘录；
- `figures/`：需要单独检查的图像；
- `data/`：允许进入知识库工作流的数据快照；
- `personal/`：用户个人写作与研究记录；
- `books/`：专著或合法保存的章节资料。

正式摄入时不修改原文件，而是在 `knowledge/sources/` 建立来源页并记录 SHA-256。

补充公共仓库边界：

- `raw/` 主要服务于本地研究工作流和目录结构管理；
- `.gitkeep` 仅用于保留空目录；
- 不提交出版社 PDF 全文、未授权书籍、私人数据或未授权第三方材料；
- 外部贡献者不要向 `raw/` 添加大型二进制文件；
- 可公开的自有材料应由维护者明确确认后再提交。
