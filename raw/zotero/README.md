# Zotero Export Inbox

此目录由用户和 Zotero/Better BibTeX 管理，Agent 只读。

推荐把 Zotero Collection `Wiki Inbox` 自动导出为：

- `wiki-inbox.bib`，或
- `wiki-inbox.json`

不要把 `zotero.sqlite` 复制到这里。

补充说明：

- `wiki-inbox.bib` 是 Zotero 导出的书目元数据快照；
- 它为 Wiki source pages 提供 citation keys；
- 它不包含、也不授权被引用文献全文；
- 它由维护者在 source 审核后手动同步；
- Codex 不得在无关任务中顺手 stage 或重写该文件；
- citation key 修改必须同步检查所有 Wiki 引用；
- 本轮 public-repository metadata 任务是一次用户明确授权的 bibliography synchronization。
