# Low-Spin Wobbling Controversies

## Motivation

本项目把 `135Pr` 与 `187Au` 两个已建立的核素级争议页放入同一证据框架，追踪 reported low-spin wobbling bands 的观测基础、识别判据、竞争解释与证据缺口。目标不是把两个案例合并成同一个物理结论，而是检验一套低自旋 wobbling identification paradigm 在不同质量区、组态和实验分析中是否保持一致、可审计且可反驳。

本页是跨案例证据导航层，不是论文草稿、领域完整综述或最终裁决。`135Pr` 与 `187Au` 的数值、能带和模型结果必须回到各自 project、source 与 raw；跨案例的“共同点”不构成第三份独立证据。

## Research Question

在不预设候选带已经是 wobbling 的前提下，`135Pr` 与 `187Au` 是否显示出一组可跨案例复用的识别判据、共同失效模式和区分性证据缺口？这些内容能否指导未来对 low-spin wobbling approximation、connecting-transition character、signature-partner 对照和替代解释的检验？

## Why Low-Spin Wobbling Identification Matters

- Wobbling motion 被用作三轴形变（triaxial deformation）和三轴转动动力学的重要 fingerprint，但“存在三轴形变”“出现候选 wobbling 能带”和“满足 wobbling 的电磁判据”是三个不同层级，不能循环论证。
- 经典 wobbling-phonon 图景通常依赖高自旋、小角进动近似。若低自旋体系不满足该近似，候选带即使具有相近能量行为，也可能需要不同的集体或单粒子语言描述。
- 在两个案例采用的 wobbling identification paradigm 中，候选 wobbling band 到参考带的 `ΔI=1` connecting transitions 需要以 E2 为主，通常对应大 `abs(δ)` branch；因此它是必要实验判据之一。mixing-ratio branch 的选择会直接改变 E2 fraction、相对 transition strength 和 wobbling assignment。
- 大 `abs(δ)` 只表示候选结构通过这项必要判据，并不单独证明 wobbling。Guo 2022 指出 β/γ vibration、tilted precession 等其他结构或集体机制也可能产生增强 E2 links。若进一步讨论 γ-softness，需要独立的形变与能谱判据；γ-softness 是势能面/形变动力学性质，不是由大 `δ` 直接标记的另一种跃迁机制。
- Angular distribution/correlation、linear polarization、mixing ratio、coincidence/level-scheme assignment 与 signature-partner 对照不是彼此可替代的“加分项”，而是识别链中约束不同歧义的实验环节。
- 如果 reported low-spin wobbling 被误判，受影响的不只是一条 band label；由此延伸的 triaxiality、collective motion、phonon hierarchy、angular-momentum geometry 与模型参数解释也需要重新评估。反之，发现分析歧义也不等于 wobbling 已被排除，只说明原有证据强度需要重新定级。

以上是本项目的研究动机与综合框架，不是对任一具体核素的最终物理结论。

## Current Hypotheses

- H1：两个案例共享的核心争议来自 connecting-transition E2/M1 character 与 mixing-ratio branch selection，而不是 wobbling 判据定义本身。
- H2：level/band identity、polarization magnitude 与 signature-partner comparison 是判据链的前置约束，缺一会降低 assignment 的可辨识性。
- H3：能量趋势与模型再现提供条件性支持，但不能覆盖冲突的 electromagnetic evidence。
- H4：low-spin approximation 的适用边界可能是跨案例解释分歧的重要理论来源；Lawrie 2020 已提供经审核的直接理论框架，但不足以把任一案例直接改判为 TiP。
- 当前状态：H1-H3 有两个已审核 case project 的导航支持；H4 已接入 theory source，新的跨来源 theory notes 仍待审核。本页不作最终取舍。

## `135Pr` Controversy Summary

### Sources and structures

- Supporting sources：Matta 2015 把 [[135pr-yrast-band]]/[[135pr-side-band]] 解释为 transverse-wobbling pair；Sensharma 2019 把 [[135pr-second-side-band]] 解释为 two-phonon extension。
- Counter-source：Lv 2022 用独立 JUROGAM II 数据、联合 `P-R_ac` 与重组后的 level scheme 反对 one-/two-phonon wobbling 指认。
- Key bands/transitions：747、813、755 keV 为 side band→yrast links；450 keV 是重组后 band 4→band 3 的关键 link。Sensharma 的 551、519、572 keV links 与弱跃迁 polarization coverage 仍影响 two-phonon chain 的闭合程度。

### Evidence and models

- Supporting observables：Matta M15-4/M15-5 报告 747/813/755 keV links 的大 `abs(δ)`、E2-rich character 与部分正 polarization asymmetry；Sensharma S19-4 报告 TW2→TW1 links 的大 `abs(δ)`。
- Counter-observables：Lv L22-4/L22-5 对 747/813/450 keV 得到小 `abs(δ)` 与约 82%/88%/91% magnetic character；L22-14 补充选门和污染检查。
- Model comparison：Matta 使用 TAC 与修改 QTR 支持 transverse geometry；Sensharma 使用 QTR/TPSM 讨论多声子能量和 transition ratios；Lv 的 standard-input、unfrozen QTR 给出 single-particle realignment/tilted-precession interpretation。上述均为模型结果或作者解释，不是实验事实。
- Unresolved issues：相同 transitions 的两组 `δ` 解需要统一 detector response、polarization convention、alignment/attenuation 与误差处理；弱 links 缺完整 polarization；缺统一 lifetime/absolute `B(E2)`；TW2 level grouping 与 one-phonon 前提并非独立。

详细 locator 与证据门见 [[135pr-wobbling-controversy]]。

## `187Au` Controversy Summary

### Sources and structures

- Supporting source：Sensharma 2020 把 Band 1/Band 2 解释为 `n_w=0/1` longitudinal-wobbling pair，并把新建 band (3) 解释为 associated unfavored signature partner。
- Counter-source：Guo 2022 用独立 HIRFL 数据、联合 `R_ac-P` 与 QTR 提出 dominant single-particle excitation 重解释，并质疑 reported band (3) 的 `187Au` identity。
- Key bands/transitions：Band 2→Band 1 的 376、462、544、639 keV links；reported band (3)→yrast 的 265、437 keV links；429 keV crossover 与相近 `188Pt` sequence 参与 band identity 争议。

### Evidence and models

- Supporting observables：AU20-4 报告四条 Band 2 links 的大 `abs(δ)` 与约 87.7%-93.3% E2 fractions；AU20-5 以近纯 M1 的 band (3) links 建立同核素 signature-partner 对照；AU20-9 报告 increasing `E_wobb`。
- Counter-observables：G22-7 对相同最低 376/462 keV links 得到约 `δ=-0.26/-0.28` 的小 branch；G22-18/G22-19 补齐 `R_ac`、polarization、选门、污染和误差处理。G22-3 未复现 reported band (3)，并把相近 sequence 归入 `188Pt`。
- Model comparison：Sensharma 的 PRM/CDFT 支持 LW/SP 分类框架；Guo 的 QTR 把 Bands 1/2 主要联系到不同 `h9/2` orbitals，并在低自旋给出较小 precession-like component。模型输入不同，不能当作 common-input benchmark。
- Unresolved issues：376/462 keV 的完整 `χ²(δ)` 与 polarization exclusion power 尚未统一重建；544/639 keV 缺对等 polarization；reported band (3) identity、early internal-conversion/β-decay 原始证据、absolute `B(E2)` 与 low-spin approximation 的定量边界尚未闭合。

详细 locator 与证据门见 [[187au-longitudinal-wobbling-controversy]]。

## Shared Points of Dispute

1. **Mixing-ratio branch selection**：两案例的核心 transitions 都存在支持方大 `abs(δ)` 与反方小 `abs(δ)` 的冲突。
2. **Polarization 的使用层级**：需要区分 sign、magnitude、calibration、gate contamination 与统计/系统误差，而不是把“测过偏振”视为单一布尔条件。
3. **Band identity before interpretation**：level scheme、coincidence 和跨反应道污染决定候选序列是否是同一核素、同一带；解释不能先于对象身份。
4. **Signature partner 与 wobbling band 的区分**：E2-rich/M1-rich decay topology 是重要比较，但该比较依赖对照带身份、组态相近性和一致的数据处理。
5. **Energy/model agreement 的证据等级**：`E_wobb`、energies 和 transition ratios 的模型再现提供一致性，不能消除相反的实验 branch。
6. **Low-spin approximation**：是否能把低自旋进动直接量子化为高自旋、小角 harmonic wobbling phonon，是跨案例理论前提，而不是由任一能带标签自动保证。
7. **Evidence independence**：同一论文中的能量、transition ratios 和模型拟合属于一条证据链；后续论文继承先前 band assignment 也不形成完全独立的 phonon hierarchy 证据。

## Evidence Gaps

- 两案例尚无统一的 raw-spectrum-to-`δ` 重分析；缺共同的 detector response、polarization convention、alignment/attenuation 和 uncertainty schema。
- 关键弱 links 的 polarization coverage 不完整，限制对整条候选 wobbling decay pattern 的外推。
- 缺可统一比较的 lifetime 与 absolute `B(E2)` 数据，现有 E2 fractions/relative ratios 部分依赖争议中的 `δ` branch。
- `135Pr` 仍需 Garg/erratum、Sensharma supplementary/full analysis 与弱 links 原始材料。
- `187Au` 仍需 Sensharma full coincidence/systematics、Rupnik β-decay、Bourgeois internal-conversion 和相关 lifetime 原始来源。
- Low-spin approximation、precession 与 TiP 的理论边界已由经审核的 Lawrie 2020 接入；其 frozen-`j` 与参数覆盖限制仍保留。
- Lawrie 2020 的 frozen-`j` 分析未穷尽 unfrozen realignment、不同 MoI ordering 或所有 transverse configurations。
- 尚缺 TiP 对 `187Au` 的直接 calculation，以及能把 TiP、wobbling、signature partner 和 single-particle excitation 同时区分开的实验 observable 组合。
- 当前仅有两个详细案例，不能据此声称所有 reported low-spin wobbling bands 都具有相同问题。

## Risks and Blockers

- 两个案例的装置、反应、角度分组和 polarization response 不同，未经统一重分析不能直接比较数值优劣。
- 关键弱 transitions、lifetime/absolute `B(E2)` 与原始 follow-up sources 缺失，使部分争议链无法闭合。
- 将两个反方 source 并列不等于形成领域共识；它们各自的模型解释仍依赖输入和近似。
- TiP 理论基础已摄入但待审；`135Nd` 原始案例和尚未覆盖的 TiP 应用仍使模型景观不完整。
- 新增 LSW-THEORY-1 至 LSW-THEORY-5 与更新后的 LSW-SYN-5 尚未人工审核，不能进入 paper evidence gate 或替代下位 project/source。

3. phonon energy/transition-probability quantization；
4. transverse/longitudinal coupling 的理论差异。

下一步仍需摄入 `135Nd` 原始实验/模型案例，并检查 TiP 是否对 `135Pr` 或 `187Au` 给出直接、可检验的 calculation。Lawrie 2020 source 已审核，但本项目仍不把 `135Pr` 或 `187Au` 改判为 TiP。

## Data-Analysis Bridge for User's Future Work

当前没有写入用户尚未提供的数据结果。未来若对任一案例重分析，建议每条关键 transition 使用同一记录 schema：

| Field | Required record |
|---|---|
| Identity | nucleus、initial/final band、`Eγ`、spin/parity、source dataset |
| Spectrum | raw/background-subtracted spectrum、gate energy/width、coincidence condition、contaminants |
| Geometry/response | detector rings、efficiency normalization、attenuation/alignment 或 `σ/I`、finite-opening correction |
| Polarization | perpendicular/parallel counts、asymmetry correction、sensitivity `Q`、sign convention |
| Mixing ratio | full fit/likelihood or `χ²(δ)`、all branches、confidence interval、error propagation |
| Structural consequence | E2 fraction、relative/absolute strength 的计算链与外部输入 |
| Interpretation layer | wobbling、signature partner、single-particle/TiP 等候选解释分列 |
| Reproducibility | script/version、fit settings、difference from published analysis |

跨案例比较应先在各核素内闭合，再比较判据表现；不得把不同核素的 transition 数值直接合并拟合。

## Innovation Candidate Notes

以下仅是研究创新候选，不是已确认创新点：

1. **Common branch-selection audit**：以统一误差和响应框架重建 `135Pr` 747/813/450 keV 与 `187Au` 376/462 keV 的大/小 `δ` branches。
2. **Identification-chain robustness**：量化 level identity、polarization coverage、signature-partner reference 和 lifetime 缺失分别会把 wobbling 证据强度削弱到何种程度。
3. **Low-spin applicability map**：在原始理论摄入后，按 `J`、valence-particle alignment、rotor angular momentum 与 deformation 建立 wobbling approximation 的适用边界。
4. **Triaxiality inference audit**：区分由独立形变观测支持的 triaxiality，与由候选 wobbling assignment 反推的 triaxial collective motion，避免循环证据。
5. **Cross-case discriminating observables**：寻找同时能区分 wobbling、signature partner、single-particle excitation 与 TiP 的最小观测量组合。

每一候选都需要新增原始来源、明确可检验预测或用户数据结果后，才可进入 paper evidence gate。
