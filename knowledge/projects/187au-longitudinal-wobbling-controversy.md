# `187Au` Low-Spin Longitudinal-Wobbling Controversy

## Project Purpose

跟踪 `187Au` low-spin bands 是否应解释为 longitudinal-wobbling bands，以及如何与 associated signature partner 区分。本轮只建立 Sensharma 2020 支持链，不裁决未来反证。

## Research Question

bands (1)/(2) 是否满足 `n_w=0/1` longitudinal-wobbling 判据？band (3) 是否可由 connecting-transition character 稳健识别为 signature partner？

## Current Hypotheses

- H1：bands (1)/(2) 是 `n_w=0/1` LW pair；
- H2：band (3) 是 yrast 的 unfavored signature partner；
- H3：后续分析可能对 mixing ratios、band assignments 或模型几何提出替代解释；
- 当前状态：只录入 H1/H2 的 supporting source。

## Analysis Status

已建立首篇支持方证据容器；尚未摄入反方 source，未形成裁决性 synthesis。

## Data-Analysis Bridge

后续如重新拟合 angular distributions，应记录角度环、效率修正、alignment、attenuation coefficients、mixing-ratio branch、误差面和与 Fig.3 的逐项差异。

## Decisions

- Sensharma 2020 仅作为 supporting source；
- LW/SP assignment 与 observed transitions 分层记录；
- increasing `E_wobb` 不脱离 E2/M1 links 和模型前提单独裁决。

## Risks and Blockers

- PRL 未给出完整 coincidence 与 angular-distribution 系统误差；
- connecting transitions 缺 linear polarization；
- PRM 的形变、转动惯量和 state mixing 具有模型依赖；
- 当前只有支持方 source。
