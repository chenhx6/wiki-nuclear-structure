# `187Au` Longitudinal-Wobbling Candidate Band

## Stable Identity

Sensharma 2020 Fig.2 的 band (2)。本页保留稳定 band label；`n_w=1` 与 LW 为作者解释。

## Level and Transition Data

本带通过六条 ΔI=1 transitions 连接 [[187au-yrast-band]]。四条最低 links 为 376.3、461.8、544.3、639.3 keV。

## Spin-Parity Evidence

部分低能级自旋宇称来自早期工作；本文以 angular distributions、coincidence 与 in-band stretched-E2 character 扩展赋值，完整细节留给后续论文。

## Configuration Assignment

作者在 PRM 中将本带与接近 half-filled `h9/2` proton 耦合 triaxial rotor 的 wobbling excitation 联系。

## Proposed Interpretation

作者依据 links 的 87.7%-93.3% E2 fractions、增加的 `E_wobb` 和 PRM 比较，将本带解释为 `n_w=1` longitudinal-wobbling band。

## Discriminating Observables

- 四条最低 LW→yrast links 的大 `abs(δ)` 和 E2 fractions；
- LW→yrast 相对 `B(E2)_out/B(E2)_in` 最大约 0.7；
- 随自旋增加的 `E_wobb`；
- 与 [[187au-signature-partner-band]] 的近纯 M1 links 对照。

## Competing Interpretations

本带此前曾被解释为 yrast 的 unfavored signature partner。当前 project 只摄入 Sensharma 2020 的重新指认，尚未摄入后续反证。
