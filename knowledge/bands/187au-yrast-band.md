# `187Au` Yrast Band

## Stable Identity

Sensharma 2020 Fig.2 的 band (1)，最低显示能级为 `E_x=121.0 keV` 的 `9/2-` isomer。

## Level and Transition Data

Fig.2 显示从 `9/2-` 至 `45/2-` 的 stretched-E2 sequence。333.8 与 413.7 keV transitions 被用于验证 angular-distribution analysis。

## Spin-Parity Evidence

bandhead 和部分低能级的自旋宇称来自早期工作；本文用已知 stretched-E2 transitions 检查 angular-distribution 方法。

## Configuration Assignment

作者以接近 half-filled `h9/2` shell 的 proton 与 triaxial rotor 耦合描述本带。

## Proposed Interpretation

作者把本带作为 `n_w=0` reference band；该标签是 [[187au-longitudinal-wobbling-controversy]] 的争议前提。

## Discriminating Observables

- band 内 stretched-E2 sequence；
- 与 LW/SP candidates 的 ΔI=1 linking-transition character；
- 作为 `E_wobb` 与 transition-ratio 分母的 reference assignment。

## Competing Interpretations

本带的存在和主要 E2 sequence 属于实验结构；`n_w=0` 与 PRM `h9/2` reference 的对应为作者解释。
