# `163Lu` SD band 1

## Stable Identity

[[domscheit-1999-triaxial-superdeformation-163lu]] 中标为 SD1 的已知超形变带。论文把它延伸并首次稳固连接到正常形变态。

## Level and Transition Data

- 带头 `13/2+`；Fig.2 延伸至 `77/2+`；
- 低端带内跃迁 196.8、263.1、314.9、386.2 keV；
- SD-ND linking transitions 为 426.5、529.4、697.1 keV；
- SD1 约占 `163Lu` 反应道强度的 10%；
- 完整能级和 γ 线见 source 页 DO99-1 至 DO99-3、原文 Fig.2 与 Table 1。

## Spin-Parity Evidence

作者依据 Table 1 的 DCO/多极性指认和到已知正常形变结构的连接，给出 `13/2+` 带头。该新指认仍待用户逐项核对 source claim DO99-2、DO99-3。

## Configuration Assignment

UC 计算将 SD1 指认为最低 `(π,α)=(+, +1/2)` 大形变组态。该量子数组合与势能面形状来自模型比较，不是由单一跃迁直接测得。

## Proposed Interpretation

实验带结构、大四极矩和转动惯量支持超形变集体转动；`γ≈±20°` 的三轴性质来自 UC/PES 极小值，因此本页保留为三轴超形变候选解释。

## Discriminating Observables

- SD-ND linking transitions 的多极性、强度和分支；
- `J^(1)`、`J^(2)` 及混合校正前后差异；
- 寿命与 [[transition-quadrupole-moment]]；
- 能区内同自旋宇称 SD/ND 态的混合。

## Competing Interpretations

低自旋 `21/2+` 处的 `J^(2)` 波动主要由 SD-ND 能级混合解释，不能直接视为形变突然变化。平均四极矩不能区分正、负 γ 组态。
