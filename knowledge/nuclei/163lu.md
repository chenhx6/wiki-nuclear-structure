# `163Lu`

## Identity

`A=163`、`Z=71`、`N=92` 的奇 A 镥核。它不属于 A≈130 质量区；本页在当前 Wiki 中用作三轴超形变与高自旋转动判据的非 A≈130 参照案例。

## Production Reactions

[[domscheit-1999-triaxial-superdeformation-163lu]] 使用 145 MeV `29Si` 轰击薄自支撑 `139La` 靶，论文将反应统写为 `139La(29Si,xn)`。形成 `168Lu` 复合核后，`139La(29Si,5n)163Lu` 产生本文核素，4n 道产生 `164Lu`；本文不讨论后者的能级与带结构。

## Established Level Structure

1999 工作把 [[163lu-sd1]] 与正常形变结构连接，确定 SD1 带头为 `13/2+`，并在 Fig.2 给出延伸到 `77/2+` 的序列。正常形变侧包含作者指认为 `[411]1/2+`、`[404]7/2+` 和 `[523]7/2-` 的带结构。

## Bands

- [[163lu-sd1]]：已连接到正常形变态，带头 `13/2+`；
- [[163lu-sd2]]：较弱的第二条 SD 带，连接跃迁未建立，最低自旋和激发能暂定；
- 正常形变 `[411]1/2+`、`[404]7/2+`、`[523]7/2-` 带本轮不单独建页。

## Deformation and Collective Features

SD1 早先寿命结果给出 `Q_SD=10.7(7) b`；1999 文献的两带混合分析给出 `Q_ND/Q_SD=0.48(7)`，并显示 SD1、SD2 有相近转动惯量。UC/PES 在大形变处给出 `γ≈+20.6°` 和 `-15.0°` 极小值；这些 γ 值是模型结果，不是直接实验测量。

## Competing Interpretations

SD2 可被解释为不同 `(π,α)` 的三轴 SD 组态，也被作者提出为 wobbling 候选。由于 SD1-SD2 linking transitions 及其多极性未建立，wobbling 在该文中没有得到实验识别。

## Key Observables

- SD1/SD2 的 `J^(1)` 与 `J^(2)`；
- SD1 的带内和 SD-ND linking γ 跃迁及 DCO；
- 两带混合参数与带外/带内分支；
- Ref.[2] 寿命测量给出的 SD/ND 四极矩。

## Open Questions

- SD2 的确定激发能、自旋宇称和 SD1-SD2 linking transitions 是什么？
- linking transitions 的 M1/E2 多极性及绝对强度是否支持 wobbling？
- 独立观测能否区分正、负 γ 极小值及 γ-soft/γ-rigid 动力学？
