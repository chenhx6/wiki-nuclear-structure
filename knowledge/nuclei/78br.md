# `78Br`

## Identity

`Z=35`、`N=43` 的 odd-odd nucleus；Liu 2016 研究其正/负宇称双带、MχD 候选与 octupole correlations。

## Production Reactions

- 60/65 MeV `70Zn(12C,p3n)78Br`：[[ithembalabs-afrodite-diamant-78br-c12-60-65mev]]。

## Established Level Structure

[[liu-2016-octupole-correlations-multiple-chiral-doublet-bands-78br]] 新增 Bands 2、4、5、44 条 transitions 与 21 levels；Fig.1 给出五条带。具体 spin/parity 依赖 ADO、偏振和 linking relations。

## Bands

- [[78br-positive-parity-mchd-pair]]：Bands 1–2。
- [[78br-negative-parity-mchd-pair]]：Bands 3–4。
- Band 1 与 Band 3 之间观测八条 E1 links。

## Deformation and Collective Features

MDC-CDFT/TPRM 给出组态相关三轴形变与 nonplanar geometry；`beta20-beta30` PES 沿 octupole 坐标较软。这些是模型结果，不是直接形变测量。

## Competing Interpretations

当前 source 支持 octupole correlations/softness，但与 `224Th` stable-octupole benchmark 明显不同；Band 3/4 configuration 为 tentative。作者认为 MχD 与 octupole correlations 共存指向未来在单核中观测 quartet 的可能性，但没有把当前四条带直接指认为已建立 quartet。

## Key Observables

ADO、linear polarization、E1 links、`B(E1)/B(E2)`、`delta E`、`S_chiral(I)` 和 `B(M1)/B(E2)`。

## Open Questions

- 能否用 lifetimes/absolute transition strengths 独立加强两对带的 chirality 与 octupole evidence？
- Band 3/4 中 `f5/2-p3/2` mixing 如何影响 configuration 和 TPRM deviations？
