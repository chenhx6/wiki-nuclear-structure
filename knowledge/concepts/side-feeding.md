# 旁路馈入（Side Feeding）

## Definition

Side feeding 指目标能级除带内 cascade feeding 外，由带外 neutron/gamma cascade 或其他路径馈入的 population component。它可改变 emitting level 的 magnetic-substate population，从而影响 γ 射线角分布。

## Source-Level Meaning

[[draper-1970-gaussian-substate-side-feeding]] 的示意图把每个 quasirotational band level 同时接受前一条 stretched-E2 cascade feeding 与 intensity `IJ` 的 side feeding。作者用 Gaussian 或 Gaussian mixture 描述 side-feeding substate population，并用 transition intensities 约束 side-feeding contribution。

## Relevance to P-ADO

若 mixing-ratio extraction 假定固定 alignment width，但实际 feeding path 与 side-feeding intensity 变化明显，拟合得到的 `δ` 可能携带该 alignment assumption 的模型依赖。Side feeding 因此是 `σ/I` 预设困难的核心物理来源之一。

## Necessary Assumptions

需要明确被分析能级、主 cascade、side-feeding intensity 和 side-feeding substate distribution 的模型形式。

## Discriminating Observables

Transition intensity balance、angular-distribution coefficients、cascade placement 和 feeding-path systematics 是主要约束。

## Supporting Evidence

Draper 1970 在 stretched-E2 formalism 中显式引入 side feeding intensity 与 Gaussian or Gaussian-mixture substate population。

## Counter-evidence and Competing Interpretations

如果 feeding path 未闭合或多个 components 混合，单一 Gaussian side-feeding description 可能只是近似而非唯一解释。

## Our Current Position

Side feeding 是 alignment uncertainty 的必要检查项；本页不提供通用数值 prior。
