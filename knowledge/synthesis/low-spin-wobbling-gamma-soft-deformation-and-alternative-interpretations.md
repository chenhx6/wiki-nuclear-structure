# Low-Spin Wobbling、γ-Soft Deformation 与替代解释

## Question and Scope

本页是在已审核 source claims 基础上建立的阶段性跨主题研究地图。它连接 [[low-spin-wobbling-controversies]]、[[gamma-soft-deformation-evidence-map]]、[[135pr-wobbling-controversy]] 与 [[187au-longitudinal-wobbling-controversy]]，但不替代这些 project、source 或 raw 原文，也不裁决任一 reported low-spin wobbling assignment 的最终身份。

## Evidence Matrix

| Layer | Core source basis | Direct contribution | Boundary |
|---|---|---|---|
| `135Pr/187Au` controversies | M15/S19/L22；AU20/G22 | 能带、connecting transitions、`δ`/偏振冲突与模型解释 | 两核与各数据集必须分别审计 |
| TiP | LAW20-1 至 LAW20-15；LV21-1 至 LV21-18 | 完整 3D rotation、wobbling approximation 条件与 `135Nd` comparison case | framework/case 不自动改判其他核素 |
| γ-soft background | ZC91；NOM17；NOM21；B19 | 低能判据、PES/shape-phase transition、pairing-triaxial coupling、随自旋形变变化 | 多数为偶偶或模型层；不能直接给 odd-A band identity |
| IBFM alternative | NOM22-1 至 NOM22-15 | γ-soft core + odd fermion 的 spectra 和 E2/M1 predictions | 理论 challenge，不是新实验或最终裁决 |

## Motivation

Wobbling motion 常被用作三轴转动动力学的重要 fingerprint；但“核具有三轴形变背景”“存在相邻低自旋带”“连接跃迁满足 wobbling 判据”和“该带可量子化为 wobbling phonon”是不同层级。低自旋、正常形变或 γ-soft 核中的角动量涨落、奇粒子重取向、配对、带混合和有限统计，会使这些层级更难分离。

如果候选带到参考带的 `ΔI=1` connecting transitions 的 E2/M1 character 或 mixing-ratio branch 不稳，wobbling assignment 的必要实验链会削弱。若相同低能带可由 TiP、IBFM、signature partner 或 single-particle excitation 等机制描述，则需要重新定级的是原有解释强度，而不是把所有 low-spin wobbling 一概判错，也不能由此反向否定核的三轴形变可能性。

## Synthesis

### Conceptual Layers

1. **Wobbling motion**：在所讨论范式中，候选 wobbling band 到 reference band 的 `ΔI=1` links 应以 E2 为主；在 harmonic-phonon 语言下还应检查低自旋近似、能量与 transition-probability quantization（M15-8；AU20-8；LAW20-3/LAW20-14）。
2. **Low-spin candidates**：`135Pr` 与 `187Au` 都存在相同关键 transitions 的大/小 `abs(δ)` 分支冲突；band identity、偏振覆盖和 signature-partner 对照也参与证据链（M15-4/L22-4 至 L22-5；AU20-4/G22-7）。
3. **γ-soft deformation background**：γ-soft 指势能沿 γ 方向较平坦、允许较大 γ 涨落；它不是固定 γ 值，也不是 wobbling 的同义词。A≈130 的 EDF/IBM 计算和低能谱判据把这种背景具体化，但不能单独决定奇 A 高自旋带身份（ZC91-1 至 ZC91-5；NOM17-1 至 NOM17-9；NOM21-1 至 NOM21-7）。
4. **TiP alternative**：Lawrie 2020 把完整三维转动写成 tilted-precession bands，并把 harmonic wobbling 视为满足 `f(n,I)<<1` 等条件时的受限近似；`135Nd` 提供由实验带结构、电磁性质和 QTR comparison 共同支持的 TiP 案例（LAW20-1 至 LAW20-14；LV21-3 至 LV21-18）。
5. **IBFM alternative**：Nomura 2022 以 γ-soft IBM core + odd fermion 在 laboratory frame 计算低能谱和 E2/M1 observables，无需先验 wobbling-phonon label；这是与 TiP/QTR 不同的替代理论链（NOM22-2 至 NOM22-14）。
6. **Experimental gate**：能级纲图、符合关系、角分布/角关联、DCO/ADO、线偏振、mixing ratio、寿命和绝对/相对跃迁强度必须分层组合；任何单项都不足以闭合身份判断。

## Shared Experimental Criteria

| Criterion | Direct role | What it can establish | What it cannot establish alone |
|---|---|---|---|
| Level scheme、gated spectra、γγ coincidence | observed fact | 固定 nucleus、band、transition 与 decay topology | 不能给出 wobbling/TiP/IBFM 身份 |
| Angular distribution / angular correlation / DCO / ADO | experimental criterion | 约束 `ΔI`、multipolarity 与可能的 `δ` branches | 未结合 polarization 和完整 fit landscape 时不能唯一选择 branch |
| Linear polarization | experimental criterion | 区分 electric/magnetic character，并帮助排除 `δ` branch | sign 或少量弱线不能代表整条带 |
| `δ(E2/M1)` 与 `abs(δ)>1` | experimental criterion | 在本页所讨论 convention 下检验 connecting link 是否 E2 predominant | 大 `abs(δ)` 是必要但非充分条件；TiP/γ vibration 也可能有增强 E2 |
| `B(E2)out/B(E2)in` | experimental/model comparison | 比较 interband collectivity 与带内 E2 尺度 | 由争议 `δ` 和外部强度导出的比值不是独立 absolute `B(E2)` |
| `B(M1)out/B(E2)in` | experimental/model comparison | 约束 magnetic decay 与 odd-particle contribution | 不应与 E2 ratio 混写；模型 agreement 不等于 observed fact |
| Lifetime、absolute `B(E2)` / `B(M1)` | observed fact after full analysis | 提供较直接的集体性和矩阵元约束 | 仍需组态、几何和带身份解释 |
| `E_wobb` 与 band-energy staggering | derived observable | 检查候选带能量组织、signature 和模型趋势 | 趋势依赖带标号，不能覆盖冲突的电磁证据 |
| Signature-partner comparison | structural control | 在相近组态下比较 M1/E2 decay topology 与 signature splitting | 对照带身份不稳时判别链同步失效 |
| γ-band staggering / PES / shape invariants | shape-background diagnostic | 约束 γ-soft 与 γ-rigid 倾向 | 不直接识别 odd-A low-spin wobbling band |

## Model / Interpretation Landscape

| Framework | What it contributes | Evidence boundary |
|---|---|---|
| PRM / QTR | energies、wave functions、transition strengths、angular-momentum geometry | 输入的 MoI、形变、pairing、orbitals 与 frozen/unfrozen `j` 会改变结论 |
| Harmonic wobbling approximation | phonon energy 与 transition-probability scaling | 只有在小分量条件实际满足时才可从完整 3D rotation 约化 |
| TiP | 完整 3D precession / coupled-rotation language | TiP calculation 或 M1-dominated links 均不能单独确认实验身份 |
| EDF-mapped IBM / IBFM | γ-soft core、odd-fermion coupling、实验室系能谱和 E2/M1 observables | PES mapping、截断模型空间、逐核拟合耦合与 M1 operator 带来模型依赖 |
| IBM / collective Hamiltonian | γ-soft/γ-rigid 极限、形状相变、γ 与配对振动 | 主要约束低能集体背景；不能直接外推到每条奇 A 高自旋带 |
| EDF-constrained mean field / PES | `(β,γ)` 势面、极小与 softness | PES 是模型结果；浅极小不等于实验直接测得固定形状 |
| Pairing + triaxial vibrations | 低激发 `0+`、γ-band energies 与 E2 strengths 的耦合 | 配对和 γ 贡献受 configuration mixing 影响，不能逐态唯一拆分 |
| Signature partner / single-particle excitation | 非 wobbling 的能带组织候选 | 需要相近组态、signature splitting、电磁 decay 和 band identity 联合检验 |

## Established Layers and Their Limits

### Observed facts

- `135Pr`、`187Au` 支持方与反方分别报告了关键 connecting transitions 的能量、角分布/角关联、偏振或 mixing-ratio 结果；这些结果在相同 transitions 上存在冲突，应逐 source 使用 locator（M15-2 至 M15-7；L22-2 至 L22-8；AU20-2 至 AU20-7；G22-2 至 G22-14）。
- `135Nd` 的 D1/TiP1/TiP2 level scheme 与关键 links 的 predominant-M1 character 是实验分析结果；TiP 身份不是直接观测（LV21-3 至 LV21-11）。
- `136Sm` 的寿命、`Q_t` 与带交叉是实验层；γ-soft→稳定三轴是模型支持的作者解释（B19-1 至 B19-5 对比 B19-7 至 B19-12）。

### Experimental criteria

- Predominant-E2 `ΔI=1` links / 大 `abs(δ)` 是当前所讨论 wobbling identification paradigm 的必要条件之一，但不是充分条件（M15-8；AU20-8；NOM22-1；LAW20-13）。
- 角分布/角关联、偏振、符合关系与寿命约束不同歧义，不能互相替代。

### Model results

- Lawrie 2020 的 `f(n,I)`、TiP/QTR 与 wobbling-equation comparisons；Nomura 2022 的 IBFM spectra、mixing ratios 与相对 E2/M1 strengths；Nomura 2017/2021 与 Babra 2019 的 PES/IBM/TRS/TPSM 结果都属于模型层。
- 跨模型相似的能量再现不构成独立实验确认，也不自动表示模型自由度物理等价。

### Author interpretations

- 支持方把部分带解释为 transverse/longitudinal wobbling；反方或 alternative sources 提出 TiP、single-particle excitation 或 IBFM-based challenge。
- Babra 2019 把 `136Sm` 解释为随自旋从 γ-soft 向较稳定三轴形状演化；这不能被改写为寿命实验直接观测到固定 γ。

### Wiki synthesis

本页的阶段性结论是：γ-softness 会扩大低能集体芯与奇粒子耦合的可用解释空间，并使固定转子近似需要额外验证；TiP 与 IBFM 分别从完整三维转动和 particle-core algebraic coupling 两条不同路径挑战“相似低能带 + QTR/能量 agreement 即 wobbling”的简化推理。该结论只组织研究问题，不给出胜负裁决。

## Counter-evidence

- 大 `abs(δ)` / E2-enhanced links 并非 wobbling 独有，但它仍是本页所讨论 identification paradigm 的必要判据之一；不能因 TiP/IBFM alternative 存在就忽略支持方电磁结果。
- `135Nd` 的 M1-dominated TiP links 不能证明所有 TiP 必须 M1 dominated，也不能机械迁移到 `135Pr/187Au`。
- `136Sm` 的实验只直接建立寿命、`Q_t` 与带交叉；TRS/TPSM 支持的 soft-to-rigid interpretation 不是模型无关形状测量。
- Nomura 2022 的 `127Xe` 大计算 `abs(δ)` 例外和 `133La` M1-ratio 偏差限制了 IBFM “普遍 M1 dominant” 的外推。

## Model Dependence

PRM/QTR/TiP 依赖 MoI ordering、形变、pairing、included orbitals 与 frozen/unfrozen `j`；IBFM 依赖 EDF→IBM mapping、截断费米子空间、逐核拟合 coupling 和 E2/M1 operators；γ-soft/rigid 判断依赖势面、集体坐标与模型 convention。跨模型的相同 band label 或能量 agreement 不表示自由度物理等价，必须比较 common observables，并把 model result 与实验量分开。

## Limitations and Missing Evidence

- `135Pr` 747/813/450 keV 与 `187Au` 376/462 keV 需要 common-response、common-error 的 `χ²(δ)` / likelihood 与 polarization 联合复核。
- 关键弱 links 缺完整线偏振、寿命和 absolute `B(E2)` / `B(M1)`；现有 relative ratios 常依赖有争议的 `δ` branch。
- `133La`、`127Xe`、`105Pd` 的原 reported experimental sources 尚未逐篇摄入；Nomura 2022 的二手比较不能替代它们。
- 需要在同一组形变、MoI、pairing、orbitals 和 electromagnetic operators 下比较 PRM/QTR、TiP、IBFM 与 wobbling approximation。
- γ-soft 证据主要来自偶偶低能谱、PES 或邻核背景；目标奇 A、目标自旋区的 nucleus-specific 形变约束不足。
- `136Sm` 只能提供“形变背景随自旋/带交叉变化”的证据结构，不能提供 `131Ce/133Ce/135Pr/187Au` 的具体形变结论。

## Why This Matters for Future User Work

若用户在 `19F+116Sn` 数据中看到 low-spin bands，第一阶段应建立对象身份而不是先贴 wobbling 标签：

1. 用 level scheme、双/三重 gated spectra、强度平衡和污染审计固定核素、能带及 connecting transitions；
2. 对所有关键 `ΔI=1` links 联合记录 DCO/ADO、angular distribution/correlation、polarization 与完整 `δ` 双解；
3. 将 `B(E2)out/B(E2)in` 与 `B(M1)out/B(E2)in` 分开，并明确它们是直接寿命结果还是由强度和 `δ` 导出；
4. 检查 signature partner、band crossing、alignment、energy staggering 与 `E_wobb`，但不让能量趋势覆盖电磁冲突；
5. 只有在 band identity、spin/parity、电磁判据和误差面稳定后，才比较 wobbling、TiP、IBFM 或其他模型；
6. 若要进入 paper evidence gate，还需 source/raw locator、P0/P1 人工审核、竞争解释、证据独立性与必要的 lifetime/absolute-strength 支撑。

最有用的中间产物是：标注门条件和污染的 level scheme/gated spectra；DCO/ADO 与 polarization 表；完整 `χ²(δ)` 或 likelihood 图；分开的 E2/M1 relative ratios；寿命/absolute matrix elements；band-energy staggering、alignment 和 `E_wobb` 图；以及各模型在 common inputs 下的逐 observable residuals。
